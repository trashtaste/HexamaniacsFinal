\doxysubsection{Detailed Description}
implements rocket interface and acts as a composite participant 

inherit extends \mbox{\hyperlink{classRocket}{Rocket}} class. It has a single engine\+List member which is a vector that holds pointers to the leaf engines. Overides the adding, removing and executing methods defined by \mbox{\hyperlink{classRocket}{Rocket}}.
